🚀 Technical Highlights
Abstraction: Utilizes a Paciente base class to define core medical attributes and mandatory behaviors.

Inheritance: Implements specialized logic through PacienteCovid and PacienteGeneral subclasses.

Polymorphism: Features overridden methods for dynamic Diagnosis, Treatment, and Consultation Pricing based on specific patient types.

Data Handling: Leverages java.time.LocalDate for precise age calculation and java.util.Scanner for interactive data input.

🛠️ Core Functionality
The system processes clinical and demographic data to automatically generate:

A Diagnosis based on symptoms like temperature, dry cough, and respiratory distress.

A customized Treatment plan.

The Consultation Fee, adjusted for variables such as hospitalization requirements.

Developed with a focus on robust logic and architectural efficiency.
